Events

JavaScipt es un lenguage conducido por eventos (Event driven)

Podemos escuchar o subscribirnos a algunos para realizar ciertas operaciones

Events Descipción
onchange Un elemento HTML ha cambiado
onclick El usuario hace click en un elemento HTML
onmouseover El usuario pasa el puntero encima en un elemento HTML
onmouseout El usuario deja de pasar por encima de un elemento HTML
onkeydown El usuario presiona una tecla
onload El navegador ha terminado de cargar la página
